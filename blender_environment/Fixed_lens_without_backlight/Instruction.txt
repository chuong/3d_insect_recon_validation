
1. Open (double click on): fixed_lens_with_backlight.blend
2. Install and turn on the addon: fixed_lens_camera_pose_control.py
3. Click on: 'viewport shading' on the the top for rendered view
4. Click on: setup the scene
5. Set camera number = 13
6. Set camera offset angle = 15
7. Set Camera distance = 0.2m (average_focus_distance)
8. Set Backward translation = -0.019m
9. Set Forward translation = 0.021m
10. Setup the resolution X = 4320 and Y = 2880 (if not set already)
11. Set step angle = 15 (if not set already)
12. Set stacking number 61 (if not set already)
13. Set Focal length = 65 mm
14. Set Aperture = 2.8
15. Save the blender file
16. Check all the parameters
17. Click on render
